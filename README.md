## How to use

### 1. Clone this Repository
  
    git clone https://github.com/deeptech-kmitl/ITReg.git

### 2. Install dependencies

    yarn install
    # or
    npm install
   
### 3. Run Dev Server

    yarn dev
    # or
    npm run dev
